# ðŸ•’ Weekly Timesheet & Utilization Tracker (Dash Version)

A full web version of the Timesheet app built using **Dash**, **Bootstrap**, and **Pandas**.

---

## Features

### Employee & Department Features
- Dropdown to select employee  
- Auto-fill department  
- Department-specific tasks & subtasks  

### Daily Tracking
- 7-day weekly table  
- Per-day:
  - Start/Stop timer  
  - Total hours  
  - Notes  
  - Multiple sessions per day  

### Task Management
- Add/delete tasks  
- Auto calculates daily and weekly totals  

### Persistence
- Saves data to **timesheet_data.json**  
- Loads automatically  

### Weekly Submission
- Export to Excel  
- Optional API submission
- Locks week after submitting
